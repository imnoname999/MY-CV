Về Tôi

Tôi là một lập trình viên web đầy đam mê. Tôi chuyên tạo ra các trang web hiện đại, đẹp mắt và dễ sử dụng, sử dụng HTML, CSS, JavaScript cùng các framework như React và Vue.js.

Tôi luôn học hỏi không ngừng và theo sát các xu hướng công nghệ mới nhất để cải thiện bản thân.

Kỹ Năng Của Tôi

HTML5

Xây dựng cấu trúc trang web chuẩn SEO và hiện đại.

CSS3

Thiết kế giao diện đẹp mắt với Flexbox, Grid và animation.

JavaScript

Tạo giao diện tương tác với JS và các thư viện hiện đại.

Node.js

Xây dựng backend linh hoạt, hiệu suất cao.

React

Phát triển ứng dụng web hiện đại dựa trên component.

PHP

Lập trình web động, xử lý dữ liệu hiệu quả.

Git

Quản lý phiên bản và cộng tác trên GitHub.

SASS

Viết CSS gọn gàng, dễ bảo trì bằng SASS.

Bootstrap

Thiết kế responsive nhanh chóng với Bootstrap.
